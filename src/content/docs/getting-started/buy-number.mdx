## Open Phone Numbers

Click **Phones** in the sidebar. The Phone Numbers page lists all numbers in your organization.

<img src={numberList.src} alt="Phone Numbers page showing an amber Buy Number button and an empty state message reading No phone numbers yet" />

## Search for a number

1. Click **+ Buy Number** to open the search page.
2. Enter a 3-digit US area code in the **Area Code** field (for example, `561`, `212`, or `310`).
3. Click **Search Numbers**.

<img src={numberSearch.src} alt="Buy Phone Number page with an Area Code text field and an amber Search Numbers button" />

A grid of available numbers appears. Each card shows the phone number, location, and supported capabilities (**Voice**, **SMS**).

<img src={numberResults.src} alt="Search results showing a grid of available phone numbers in area code 561 with location, Voice and SMS badges, and a Select button on each card" />

## Buy the number

Click **Select** on the number you want. A confirmation dialog appears showing the number, location, and capabilities. Click **Confirm Purchase** to complete the purchase.

<img src={numberConfirm.src} alt="Confirm Purchase dialog showing phone number (561) 810-2270, location Boynton Beach FL, Voice and SMS badges, with Cancel and Confirm Purchase buttons" />

## Assign an agent

After purchase, Loquent redirects you to the Phone Numbers list. Click on your new number to open the **Phone Details** page.

Select your agent from the **Assigned Agent** dropdown and click **Save**. All inbound calls to this number now route to the assigned agent.

<img src={numberAssign.src} alt="Phone Details page showing the purchased number, an Assigned Agent dropdown set to No agent, and a Save button" />

## Make a test call

Dial your new number from any phone. The assigned agent answers and responds based on the prompt you configured. Speak naturally and verify the agent behaves as expected.

If the agent does not respond correctly, return to the [Create an Agent](/getting-started/create-agent/) page and adjust the **Agent Prompt**.

:::tip
Your AI voice agent is live and handling calls. Revisit the agent prompt or voice settings at any time to refine the experience.
:::
