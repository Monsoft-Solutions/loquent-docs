## Open Call History

Click **Calls** in the sidebar. The Call History page displays a log of all calls made and received by your agents.

<img src="https://wxaijzyufbt0o3u0.public.blob.vercel-storage.com/calls/calls-list.png" alt="Call History page showing the heading, subtitle, and an empty state message reading No calls yet" />

When calls start flowing in, each entry shows:

| Column | Description |
|--------|-------------|
| **Phone Number** | The caller or recipient number |
| **Agent** | The agent that handled the call |
| **Direction** | Inbound or outbound |
| **Duration** | How long the call lasted |
| **Date** | When the call occurred |

Click any call to view its details, including the full transcription and any to-dos extracted by analyzers.

:::tip
Call history populates automatically once your agent is assigned to a phone number and starts receiving calls. No setup is needed beyond the [Getting Started](/getting-started/welcome/) steps.
:::
