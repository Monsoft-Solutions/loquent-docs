## Open Knowledge Bases

Click **Knowledge** in the sidebar. The Knowledge Bases page lists all knowledge bases in your organization. Each card shows the knowledge base name, the number of documents it contains, and how many agents are assigned to it.

<img src="https://wxaijzyufbt0o3u0.public.blob.vercel-storage.com/knowledge/knowledge-base-list.png" alt="Knowledge Bases page showing the Loquent Product Guide card with 3 docs and 0 agents badges, and an amber Create Knowledge Base button" />

## Create a knowledge base

Click **+ Create Knowledge Base** to open the creation form. Enter a descriptive title for your knowledge base — this is how you identify it when assigning it to agents later.

<img src="https://wxaijzyufbt0o3u0.public.blob.vercel-storage.com/knowledge/knowledge-base-create.png" alt="New Knowledge Base form with a Title text field showing placeholder text and a Save button" />

Click **Save**. Loquent creates the knowledge base and redirects you to the detail page where you can add documents.

## Add documents

The Knowledge Base Details page shows the title, a list of documents, and the assigned agents section. Click **+ Add Document** to open the document form.

<img src="https://wxaijzyufbt0o3u0.public.blob.vercel-storage.com/knowledge/knowledge-base-add-doc.png" alt="Knowledge Base Details page with the Add Document form expanded showing Title, Description, and Content fields, with existing documents listed below" />

Fill in the document fields:

| Field | Required | Description |
|-------|----------|-------------|
| **Title** | Yes | A name that identifies this document in the list |
| **Description** | No | A short summary of what the document covers |
| **Content** | Yes | The full text your agent can search during calls |

Click **Add** to save the document. Repeat this process for each document you want to include. There is no limit on the number of documents per knowledge base.

## Review your knowledge base

After adding documents, the detail page shows all documents with their titles and descriptions. You can expand any document to view or edit its content, or click the chevron to collapse it.

<img src="https://wxaijzyufbt0o3u0.public.blob.vercel-storage.com/knowledge/knowledge-base-detail.png" alt="Knowledge Base Details page for Loquent Product Guide showing three documents: Loquent Platform Overview, Loquent Pricing and Plans, and Loquent FAQ and Troubleshooting" />

## Assign to an agent

The **Assigned Agents** section at the bottom of the detail page lets you select which agents have access to this knowledge base. Agents with an assigned knowledge base can search its documents during live calls to provide accurate, context-aware responses.

:::tip
Build focused knowledge bases around a single topic (product info, pricing, troubleshooting) rather than one large base with everything. This helps agents find the right information faster during calls.
:::
