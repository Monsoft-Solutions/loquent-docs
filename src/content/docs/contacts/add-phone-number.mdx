---
title: Add a Phone Number
description: Link a phone number to a contact so Loquent can identify them when they call
---

Adding a phone number to a contact lets Loquent identify the caller automatically. When a call comes in from a linked number, the agent greets the caller by name and has access to their contact history.

## Add a phone number

Open a contact's details page and click **+ Add Phone**.

<img src="https://wxaijzyufbt0o3u0.public.blob.vercel-storage.com/contacts/contact-add-phone.png" alt="Add Phone form showing Phone Number field with +1234567890 placeholder, Label field with Mobile/Work/Home example, Preferred checkbox, and Cancel and Add buttons" />

Fill in the form:

| Field | Required | Description |
|-------|----------|-------------|
| **Phone Number** | Yes | E.164 format — include the country code (e.g. `+12125550100`) |
| **Label** | No | A label to identify the number, e.g. Mobile, Work, Home |
| **Preferred** | No | Mark this as the primary number for this contact |

Click **Add** to save the number.

## Preferred number

A contact can have multiple phone numbers. The **Preferred** checkbox marks one as the primary number. When Loquent needs to reference the contact's number (for example in call logs), it uses the preferred one.

:::tip
You can change the preferred number at any time by editing the contact and updating the Preferred setting.
:::

## Remove a phone number

On the Contact Details page, expand the phone number row and click **Remove**.
