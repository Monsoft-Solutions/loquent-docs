# Design: Loquent 0.3.0 Documentation Plan

**Date:** 2026-02-27
**Release:** 0.3.0 — February 2026
**Source PR:** [#221](https://github.com/Monsoft-Solutions/loquent-app/pull/221)

## Overview

33 PRs shipped in 0.3.0. Documentation is organized into 3 phases ordered by user impact. Each feature gets its own page (one page per feature). Docs serve both new users discovering features and existing users understanding what changed.

---

## Phase 1 — SMS Messaging + Contact Intelligence

Highest priority. Covers the two biggest user-facing changes in 0.3.0.

### New section: Messaging

| Page | Slug | What it covers |
|------|------|----------------|
| Messaging Feed | `messaging/messaging-feed` | The unified view — calls + SMS side by side, date groups, filter bar (All / Calls / Messages), navigating to a contact |
| Send an SMS | `messaging/send-sms` | Composing and sending outbound SMS, outbound vs inbound bubble appearance |

### Contacts — new pages + rewrite

| Page | Slug | What it covers |
|------|------|----------------|
| Contact Details *(rewrite)* | `contacts/contact-details` | New 3-column CRM layout: sidebar (avatar, actions, tags, phone/email), center timeline (calls + notes + todos with filter), right panel |
| Contact Notes | `contacts/contact-notes` | Adding/editing/deleting notes, markdown support, AI-generated notes labeled "AI Agent" |
| Contact Tags | `contacts/contact-tags` | Creating org tags, assigning to contacts, filtering by tag, AI auto-tagging behavior |
| Contact Memory | `contacts/contact-memory` | What memory is, how AI builds it per caller, where it shows up, how it affects agent behavior |

**Pages affected:** 2 new (Messaging), 3 new + 1 rewrite (Contacts)
**Sidebar changes:** Add new "Messaging" section

## Phase 2 — AI Agent Configuration

New "Agents" sidebar section. Agents currently only appear in Getting Started — they deserve their own section.

| Page | Slug | What it covers |
|------|------|----------------|
| Instruction Builder | `agents/instruction-builder` | Writing instructions from scratch, AI generation from plain language, edit toolbar (Improve, Add Personality, Fix Issues, etc.), diff view, custom edit |
| Organization Profile | `agents/organization-profile` | Filling out business name, hours, services, address; how "Enhance with org data" weaves it into agent instructions |
| Tool Configuration | `agents/tool-configuration` | What tool_config is, configuring transfer call / lookup caller / query knowledge per agent |

**Pages affected:** 3 new (Agents)
**Sidebar changes:** Add new "Agents" section

## Phase 3 — Calls, Dashboard, Settings

### Calls — new pages

| Page | Slug | What it covers |
|------|------|----------------|
| Cold Transfer | `calls/cold-transfer` | How agents hand off an active call to a human number, caller experience, configuring the transfer number |

### New section: Dashboard

| Page | Slug | What it covers |

Update `settings/settings` to cover:
- Phone number friendly names
- Organization hours

No new pages needed.

**Pages affected:** 2 new (Calls), 1 new (Dashboard), 1 update (Settings)
**Sidebar changes:** Add new "Dashboard" section
